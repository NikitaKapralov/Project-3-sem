# Система анализа расходов с прогнозированием бюджета

Веб-сервис для учета личных финансов, который помогает не только отслеживать расходы, но и прогнозировать бюджет на следующий месяц на основе ваших исторических данных.

**Демо-версия:** 

## Возможности

-  **Учет операций:** Внесение доходов и расходов по категориям
-  **Аналитика:** Визуализация финансовых данных в виде графиков
-  **Прогнозирование:** Автоматический расчет бюджета на следующий месяц
-  **Персонализация:** Создание собственных категорий с выбором цвета
-  **Адаптивный дизайн:** Удобный интерфейс для всех устройств

## Технологии

- **Backend:** Python 3.10+, Django 4.2+
- **База данных:** SQLite (разработка), PostgreSQL (продакшен)
- **Аналитика данных:** Pandas, NumPy
- **Визуализация:** Chart.js
- **Фронтенд:** Bootstrap 5, HTML5, CSS3
- **Деплой:** PythonAnywhere
